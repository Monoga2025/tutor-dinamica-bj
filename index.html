Integral Doble — Rectángulo
∬_R f(x,y) dA
= ∫_c^d ∫_a^b f(x,y) dx dy
= ∫_a^b ∫_c^d f(x,y) dy dx
R = [a,b]×[c,d]. Puedes cambiar el orden de integración libremente.
Integral Doble — Región Acotada
∬_R f(x,y) dA
= ∫_a^b [∫_{g₁(x)}^{g₂(x)} f(x,y) dy] dx
Los límites de y dependen de x (o viceversa). Dibujar siempre la región primero.
⚠️ El límite inferior debe ser MENOR que el superior
Área con Integral Doble
A = ∬_R 1 dA

Entre curvas: A = ∫_a^b [f_arriba - f_abajo] dx
Paso 1: Iguala las curvas → intersecciones
Paso 2: Determina cuál está arriba
Paso 3: Integra entre los puntos de intersección
Centro de Masa con Densidad ρ(x,y)
M = ∬_R ρ(x,y) dA
M_y = ∬_R x·ρ(x,y) dA
M_x = ∬_R y·ρ(x,y) dA

x̄ = M_y / M
ȳ = M_x / M
⚠️ M_y usa x (no y). M_x usa y (no x). Es contra-intuitivo.
Coordenadas Polares
x = r·cos(θ)
y = r·sin(θ)
x² + y² = r²
dA = r dr dθ
⚠️ NUNCA olvides la r en dA = r dr dθ. Sin r = error garantizado.
Círculo r≤R completo: θ:0→2π
Semicírculo superior: θ:0→π
Cuarto de círculo: θ:0→π/2
¿Cuándo usar Polares?
Señales de alerta:
√(x²+y²) en el integrando
Región circular o semicircular
x²+y² en el integrando
1/(1+√(x²+y²))
Si ves cualquiera de estas señales → cambiar a polares simplifica TODO.

σ Estadística Descriptiva

⏰ 42 horas

Medidas de tendencia central, dispersión, tablas de frecuencia, gráficas.

Medidas de Tendencia Central
Media: x̄ = Σxᵢ / n
Agrupada: x̄ = Σ(fᵢ·mᵢ) / n
Mediana: valor central (n impar)
Media de 2 centrales (n par)
Moda: valor más frecuente
mᵢ = marca de clase (punto medio del intervalo)
Mediana — Datos Agrupados
Me = L + [(n/2 - F) / f] · c
L = límite inferior de la clase mediana
F = frecuencia acumulada ANTES de la clase mediana
f = frecuencia de la clase mediana
c = amplitud del intervalo
Moda — Datos Agrupados
Mo = L + [d₁ / (d₁ + d₂)] · c
d₁ = f_modal - f_anterior
d₂ = f_modal - f_siguiente
c = amplitud
Varianza y Desviación Estándar
Población: σ² = Σ(xᵢ-μ)² / N
Muestra: s² = Σ(xᵢ-x̄)² / (n-1)
Alternativa: s² = [Σxᵢ² - (Σxᵢ)²/n] / (n-1)
σ o s = √(varianza)
⚠️ Población usa N. Muestra usa (n-1). Diferencia crítica.
Coeficiente de Variación
CV = (s / x̄) · 100%
Compara dispersión entre conjuntos de datos con unidades diferentes. CV < 15% = homogéneo.
Tabla de Frecuencias
k = √n (regla de Sturges: k = 1 + 3.322·log(n))
c = (Máx - Mín) / k
fᵣ = fᵢ / n
F = acumulada (suma sucesiva)
fᵢ = frecuencia absoluta, fᵣ = relativa, F = acumulada
Cuartiles
Posición Qₖ = k·n/4
Q1 = percentil 25
Q2 = mediana (percentil 50)
Q3 = percentil 75
RIC = Q3 - Q1
Sesgo y Curtosis
Sesgo positivo: media > mediana > moda
Sesgo negativo: media < mediana < moda
Curtosis > 3: leptocúrtica (pico alto)
Curtosis = 3: mesocúrtica (normal)
Curtosis < 3: platicúrtica (achatada)

Análisis de mallas (KVL). Tu circuito: 3 mallas, 2 fuentes 80V, resistencias 20Ω/30Ω

📋 El profe dijo: Traer el circuito dibujado 3 veces (una por malla) con colores diferentes. El quiz es guiado — él te lleva paso a paso. Solo tienes que saber plantear las ecuaciones de malla.
🧠 Método de Mallas — Lo que debes saber de memoria
Paso 1 — Asignar mallas
Dibuja corrientes circulares en
cada lazo independiente.

Tu circuito: 3 mallas
I₁ (roja), I₂ (azul), I₃ (verde)
Todas en sentido horario ✓
Convención: SIEMPRE todas en el mismo sentido (horario). Facilita los signos.
Paso 2 — KVL por cada malla
ΣV = 0 en cada lazo

Fuente: + si la corriente
   sale del terminal +
Resistencia propia: - R·Iₙ
Resistencia compartida: + R·Iₖ
⚠️ Resistencia compartida: si la otra corriente va en MISMO sentido → suma. Contrario → resta.
Paso 3 — Sistema matricial
[R]·[I] = [V]

| R₁₁ -R₁₂ -R₁₃ | |I₁| |V₁|
|-R₂₁ R₂₂ -R₂₃ | |I₂| = |V₂|
|-R₃₁ -R₃₂ R₃₃ | |I₃| |V₃|
Rₙₙ = suma de resistencias en malla n. Rₙₖ = resistencia compartida entre n y k (siempre negativa en diagonal).
Paso 4 — Resolver e interpretar
Resolver sistema 3×3
(Cramer, sustitución, o Gauss)

Si I < 0 → la corriente real
va en sentido contrario al asumido

V_R = R · I_rama
I_rama = I_malla propia ± I_malla compartida (+ si mismo sentido, - si contrario)

ELEMENTOS

Rb = 20Ω  (compartida M1-M2)
Ra = 30Ω  (compartida M1-M3)
R2 = 20Ω  (rama inferior, solo M2)
Rc = 30Ω  (compartida M2-M3)

V₂ = 80V  (fuente malla 2)
TOPOLOGÍA
M1: 80V — R1(20) — Ra(30) — Rb(20)
M2: 80V — Rb(20) — R2(20) — Rc(30)
M3: Ra(30) — Rc(30) — R3(30)
    (sin fuente en M3)
PASO 2 — Ecuaciones KVL (las 3 mallas)
Malla 1 — Suma de todas las resistencias en M1, menos las compartidas × sus corrientes:
(R1+Rb+Ra)·I₁ - Rb·I₂ - Ra·I₃ = 80V
(20+20+30)·I₁ - 20·I₂ - 30·I₃ = 80
70I₁ - 20I₂ - 30I₃ = 80    ... (1)
Malla 2 — Igual proceso:
(Rb+R2+Rc)·I₂ - Rb·I₁ - Rc·I₃ = 80V
(20+20+30)·I₂ - 20·I₁ - 30·I₃ = 80
-20I₁ + 70I₂ - 30I₃ = 80    ... (2)
Malla 3 — Sin fuente, lado derecho = 0:
(Ra+Rc+R3)·I₃ - Ra·I₁ - Rc·I₂ = 0
(30+30+30)·I₃ - 30·I₁ - 30·I₂ = 0
-30I₁ - 30I₂ + 90I₃ = 0    ... (3)
PASO 3 — Sistema matricial simplificado
|  70  -20  -30 |  |I₁|    |80|
|-20   70  -30 | · |I₂| = |80|
|-30  -30   90 |  |I₃|    | 0|
Observa: las ecuaciones (1) y (2) son simétricas → sugiere I₁ = I₂
De (3) dividida entre 30:  -I₁ - I₂ + 3I₃ = 0 → I₃ = (I₁+I₂)/3
PASO 4 — Solución del sistema
Como (1) = (2) → I₁ = I₂. Sustituye en (1):
70I₁ - 20I₁ - 30·(2I₁/3) = 80
70I₁ - 20I₁ - 20I₁ = 80
30I₁ = 80
I₁ = 8/3 A ≈ 2.67 A (horario)
I₂ = 8/3 A ≈ 2.67 A (horario)
I₃ = 16/9 A ≈ 1.78 A (horario)
Verificación: I₃ = (I₁+I₂)/3 = (8/3 + 8/3)/3 = (16/3)/3 = 16/9 ✓
VERIFICACIÓN — Comprueba las 3 mallas
Malla 1: 70(8/3) - 20(8/3) - 30(16/9) = 560/3 - 160/3 - 160/3 = 240/3 = 80 ✓
Malla 2: -20(8/3) + 70(8/3) - 30(16/9) = -160/3 + 560/3 - 160/3 = 240/3 = 80 ✓
Malla 3: -30(8/3) - 30(8/3) + 90(16/9) = -80 - 80 + 160 = 0 ✓
